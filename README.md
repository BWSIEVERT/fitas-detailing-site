# Fitas Auto Detailing Website

A premium one-page Next.js website for Fitas Auto Detailing in Dalton, GA.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`

## Production build

```bash
npm run build
npm start
```

## Deploy on Vercel

1. Push the project to GitHub.
2. Sign in to Vercel.
3. Click **Add New > Project**.
4. Import the repository.
5. Leave the framework preset as **Next.js**.
6. Click **Deploy**.
